# NuSTORM
Author: Jack Franklin, jack.d.franklin@durham.ac.uk
################################################################

This project contains some python code that should help with 
investigating the sensitivity of the NuSTORM experiment to 
physics Beyond the Standard Model (BSM). Flux and cross-section
files are provided by the NuSTORM collaboration.

################################################################

Layout of the code:

    -src
	-flux.py
	-cross_sections.py
    
    -resources
	-fluxes
	-cross_sections

    -tests
	-flux_test.py

################################################################
